# Personal-Brain-Openwiki

Welcome to the **Personal-Brain-Openwiki** repository! 

This repository is your comprehensive and practical guide to understanding and applying LangChain's **OpenWiki** tool, and how to integrate it with **DeepAgents** to build AI agents with proactive memory and deep context for your projects.

## What is OpenWiki?
OpenWiki is an open-source Command Line Interface (CLI) agent launched by LangChain. Its primary function is to generate and maintain a knowledge base (Wiki) either for code repositories (Code Brain) or as a personal memory (Personal Brain) for AI agents.

## 🚀 Quick Start

### Prerequisites
- Node.js v18+
- LLM Provider API Key (OpenAI or Anthropic)

### Installation
```bash
git clone https://github.com/ALaaAlawdi/Personal-Brain-Openwiki.git
cd Personal-Brain-Openwiki
npm install
```

### Setup
```bash
cp .env.example .env
# Edit the .env file and add your API key
```

### Running the Project
```bash
# View available commands
npm start

# Run the Code Reviewer Agent (uses OpenWiki as memory)
npm run agent:code-reviewer

# Run the Personal Wiki Memory Agent
npm run agent:wiki-memory

# Initialize Personal Brain
npm run cli:brain -- init

# Simulate data ingestion from sources (Gmail, X, Notion)
npm run cli:ingest

# Ask the Personal Brain a question
npm run cli:brain -- query "What are my active projects?"
```

## 🧠 How Does the Project Work?

### 1. Code Reviewer Agent (`code-reviewer-agent.ts`)
Demonstrates how a DeepAgent reads the documentation generated by OpenWiki (Code Brain) and uses it as context for code reviews. Instead of searching through thousands of files, the agent reads the architectural summaries first.

### 2. Wiki Memory Agent (`wiki-memory-agent.ts`)
Shows how a DeepAgent builds a local memory (Personal Brain) from Markdown files. It can search the wiki, add new information, and display all entries.

### 3. CLI Interface (`personal-brain-cli.ts`)
A full command-line application that allows you to initialize the Personal Brain and ask it questions, built on Commander.js and DeepAgents.

### 4. Ingestion Simulation (`ingest-sources.ts`)
Simulates how OpenWiki fetches data from connectors (Gmail, X, Notion) and converts them into Markdown files in the `wiki-output/` directory.
